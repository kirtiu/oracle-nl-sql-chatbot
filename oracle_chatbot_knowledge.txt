### SQL Query
List employees with salary greater than 80000  
SQL: SELECT first_name, last_name, salary FROM Employee WHERE salary > 80000;

### SQL Query
Find average salary in each department  
SQL: SELECT d.name, AVG(e.salary)  
     FROM Employee e  
     JOIN Department d ON e.department_id = d.department_id  
     GROUP BY d.name;

### SQL Query
Highest paid employee in the company  
SQL: SELECT first_name, last_name, salary  
     FROM Employee  
     ORDER BY salary DESC  
     FETCH FIRST 1 ROWS ONLY;

### SQL Query
Lowest paid employee in IT department  
SQL: SELECT first_name, last_name, salary  
     FROM Employee  
     WHERE department_id = (SELECT department_id FROM Department WHERE name = 'IT')  
     ORDER BY salary ASC  
     FETCH FIRST 1 ROWS ONLY;

### SQL Query
Find employees by department name  
SQL: SELECT e.first_name, e.last_name, d.name  
     FROM Employee e  
     JOIN Department d ON e.department_id = d.department_id  
     WHERE d.name = 'Finance';

### SQL Query
Find employees by position title  
SQL: SELECT e.first_name, e.last_name, p.title  
     FROM Employee e  
     JOIN Position p ON e.position_id = p.position_id  
     WHERE p.title = 'Software Engineer';



### FAQ
Q: Who is the highest paid employee in IT department?  
A: SELECT first_name, last_name, salary FROM Employee  
   WHERE department_id = (SELECT department_id FROM Department WHERE name = 'IT')  
   ORDER BY salary DESC FETCH FIRST 1 ROWS ONLY;

### FAQ
Q: Which department has the highest average salary?  
A: SELECT d.name, AVG(e.salary) FROM Employee e  
   JOIN Department d ON e.department_id = d.department_id  
   GROUP BY d.name ORDER BY AVG(e.salary) DESC FETCH FIRST 1 ROWS ONLY;

### FAQ
Q: How can I find all employees working as Software Engineers with salary above 80000?  
A: SELECT e.first_name, e.last_name, e.salary FROM Employee e  
   JOIN Position p ON e.position_id = p.position_id  
   WHERE p.title = 'Software Engineer' AND e.salary > 80000;

### FAQ
Q: Which position has the lowest average salary in the company?  
A: SELECT p.title, AVG(e.salary) FROM Employee e  
   JOIN Position p ON e.position_id = p.position_id  
   GROUP BY p.title ORDER BY AVG(e.salary) ASC FETCH FIRST 1 ROWS ONLY;

### FAQ
Q: How do I list employees in Finance department earning less than 60000?  
A: SELECT e.first_name, e.last_name, e.salary FROM Employee e  
   JOIN Department d ON e.department_id = d.department_id  
   WHERE d.name = 'Finance' AND e.salary < 60000;

### FAQ
Q: Can I see the top 3 highest paid employees across all departments?  
A: SELECT first_name, last_name, salary FROM Employee  
   ORDER BY salary DESC FETCH FIRST 3 ROWS ONLY;

### FAQ
Q: How do I check the number of employees in each department?  
A: SELECT d.name, COUNT(e.employee_id) FROM Employee e  
   JOIN Department d ON e.department_id = d.department_id  
   GROUP BY d.name;

### FAQ
Q: How can I find the average salary for Managers?  
A: SELECT AVG(e.salary) FROM Employee e  
   JOIN Position p ON e.position_id = p.position_id  
   WHERE p.title LIKE '%Manager%';

### FAQ
Q: Which department has the most employees earning above 70000?  
A: SELECT d.name, COUNT(e.employee_id) FROM Employee e  
   JOIN Department d ON e.department_id = d.department_id  
   WHERE e.salary > 70000 GROUP BY d.name ORDER BY COUNT(e.employee_id) DESC;

### FAQ
Q: How do I update the salary of a Sales Executive?  
A: UPDATE Employee SET salary = new_value  
   WHERE position_id = (SELECT position_id FROM Position WHERE title = 'Sales Executive')  
   AND employee_id = your_id;
